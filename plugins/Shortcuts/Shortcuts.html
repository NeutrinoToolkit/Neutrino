+ : increase zoom
- : decrease zoom
= : adapt zoom to window or set 100%
%CTRL%m : toggles mouse shape
%CTRL%r : show/hide origin ruler
%CTRL%g : show/hide grid
%CTRL%0 : set origin to mouse position
%CTRL%=Shift-0 : same but for all images
h : open the horizontal lineout
v : open the vertical lineout
%CTRL%-Up : go to previous image
%CTRL%-Down : go to next image
c : open colorbar window
%CTRL%-Left : go to previous colorbar
%CTRL%-Right : go to next colorbar
%CTRL%-A : rescale colorbar to image min/max
%CTRL%-9 : rescale colorbar to 99% of the pixels
> : increases the γ
< : reduces the γ
%CTRL%-9 : Rescale the colorbar so that 99% of the pixels are visible
i : opens the list window
Shift-i : opens the measure window
Tab : cycle over paths
w : pop up the path-property window.
Arrows will move the path
Shift+arrow will move faster
backspace : erase the path (if it's not owned by any window).
Lines:
b : toggle Bezier line
c : toggle closed line
p : prepend point
a : append point
d : delete point
x : will put the first 2 point horizontal
y : will put the first 2 point vertical
r : will put the first 4 points in square
shift-r : will make 4 points expand to the image
%CTRL%-%OPTION%-%COMMAND%-S : save a screenshot of the current window to the home folder.
%CTRL%-%OPTION%-%COMMAND%-J : copy a status of the open analysis window to be reused into the javascript console.
