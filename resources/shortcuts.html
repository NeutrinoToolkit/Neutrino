+ : increase zoom
- : decrease zoom
= : adapt zoom to window or 100%)

m : toggles mouse shape
r : show/hide origin ruler
o : set origin to mouse position
O : same but for all images

h : open the horizontal lineout
v : open the vertical lineout
Option-Up : go to previous image
Option-Down : go to next image

C : open colorbar window
c : inverse colorbar
a : rescale image colors to limits
Option-Left : go to previous colorbar
Option-Right : go to next colorbar

i : opens the measure window

Tab : cycle over paths
Once a path is selected, pressing return will pop up the path-property window. Arrows will move the path and backspace will erase the path (if it's not owned by any window).
