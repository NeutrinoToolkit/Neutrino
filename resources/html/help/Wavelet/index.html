Wavelet analysis

In the Window there are 3 tabs : Wavelet, Unwrap and Subtract. If all the parameters are already set you can press the button to get all the 3 calculation done one after each other (useful if you have a series of images to process)

Wavelet:

This is the filter panel.
  1. Select the image to filter
  2. Set the parameters for the Morlet wavelet filter, for the stretch and the angle, give the min,  max and the number of test functions.
  3. Based on the Rel. to carrier check the stretch values are relative to the carrier. When this option is active, check carefully the values for the carrier (interfringe and rotation) and if they're not right, hit the button to get the right values (you may want to do this by moving the ROI on a steady part of the interferogram or directly on a reference image).
  4. Set a proper value for the Thickness of the fringes (i.e. the transverse part of the 2D Morlet function)
  5. The Damp value should remain 1.0 unless the interferogram is really noisy and you want to calculate the phase by taking more peaks correlation.
  6. Use Crop region check if you want to get an image of the original ROI
  7. Use Synthetic to generate an artificial image of the interferogram based on the phase and contrast values.
  8. Erase previous will overwrite the previous set of result images.
  9. Hit the Calculate button to start the calculation. (Go to the preferences panel if you want to use the OpenCl algorithm)

Unwrap:

Here we do a 2D unwrap of the interferogram.

  1. Select the phase and quality images. Usually these are already set after the previous computation done, otherwise you have to manually select them.
  2. Choose the unwrap method (Quality and Miguel are usually a good choice).
  3. Choose or not to activate a barrier by checking the box right the unwrap method. This will display a barrier line which will avoid the unwap algorithm to unwrap the phase through.
  4. Choose whether or not remove the carrier plane phase.
  5. Hit the Unwrap button

Subtract:

Once you get the phase, here we subtract either the carrier plane (if not done in the previous tab) or a reference image phase.
The Offset value is subtracted to the whole image
