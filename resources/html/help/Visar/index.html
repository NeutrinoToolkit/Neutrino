Settings of the parameters


Select the Settings tab on the right and select the Visar 1 or Visar 2 tab to set the initial parameters of each VISAR (see figure 1)


1/ For each VISAR, select the reference (before the laser shot) and the shot (with the laser).


2/ Select the ROI on each images. The phase is taken from the middle of the ROI, while the intensity is averaged over the entire ROI.


3/Adjust the intensity parameters

3.1. adjust the factor to have the ref (dashed line) and the shot (full line) at the same level, to correct from the shot-to-shot fluctuations of the laser energy

3.2. adjust the background of the ref and of the shot. To estimate its value, read on the image directly on an appropriate area (long time after the shock for example)


Note: if the 2 visars are based on the same laser, the correction factor (3.1) should be the same. In general, the reference and the shot background are the same.


4/ Fit the interference fringes

4.1. adjust the rotation of the image if the fringes are not vertical

4.2. press the guess button and check manualy the estimation of interference fringes is roughly correct.

4.3. If not, adjust the frequency weight to remove either the lower frequencies (global fluctuation of the signal on the image) or the higher frequencies (small fluctuation, due to noise, on the image).


5/ Adjust the damp and resolution parameters

Those parameters are adjusting the 2D complex Morlet function, which is used to better extract the phase in the Fourier transform space (filter consisting in the multiplication of a complex exponential carrier - cosinus - with a gaussian envelope). This function allows to better correlate the fringes for an optimized extraction of the final phase.

5.1. Damp parameter is the width of the Gaussian part of the Morlet function in the horizontal direction. If >1 this is extending the area considered for the phase extraction. This is useful in case of an important noise. Usually this value is set at 1.

5.2. Resolution parameter (in px) is corresponding to the width of the Gaussian part of the Morlet function in the vertical direction, i.e. the time direction.

Extracting the velocity profile


Select the second tab on the right, untitled Velocity.


1/ For each VISAR, specify the sensitivity (Velocity/frange)


2/ Specify the Time 0 window

2.1. Pixel position of the synchronization (see synchronization shot)

2.2. Delay when compared with the initial synchronization shot (if changed)


3/Calibration of the streak camera specify the time scale in ns/pixel


4/The offset window is corresponding to a fine adjustment between the reference and the shot !!! to precise with tommaso. This is adjusted automatically. The automatic value is displayed. If needed (when the value of the velocity at negative time is not at 0), an additional offset can be set manually.


5/ The Reflectivity window allows to precise the real value of the reflectivity.

5.1. To set the final reflectivity at long time delay, after the shock, in order to obtain 0

5.2. To set the real value of the reflectivity of the used material


6/ To correct the jumps of fringes

In case of a jump, the 2 Visars won’t give the same velocity. Here the objective is to apply a correction and obtain the 2 same velocities.

6.1. first look manually to find the number of jumps of each VISAR. After this manual look, set this parameter to 0

6.2. Three parameters need to be adjusted:

  • t = time for the beginning of the velocity change (in pixel).

  • i = number of jumps to correct for (defined by step 6.1)

  • [n0] = index of refraction. This is optional. To be changed if the sensitivity is different??? See with Tommaso.

This all set of parameters is set for each jump.

As an example “0 3;109 5” in case of a first change of velocity at the time 0 with a jump of 3 followed by a second change of velocity at the time 109 with a jump of 5.
