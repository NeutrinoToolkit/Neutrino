Pyhton scripting

Neutrino uses the PyhtonQt library, for which we expose the python variable neu (the current neutrino window) and nApp (the current application, from which you can query all neutrino windows)

Syntax:


list of neutrino windows:neu_list=nApp.neus()
current displayed image
this_neu=neu.getBuffer()
alternative
this_neu=neu_lis[0]
convert image to numpy array
n_arr=phys2array(this_neu)
convert numpy array to nPhysD
new_neu=array2phys(n_arr)
display a python controlled nPhysD
neu.addShowPhys(new_neu)
open XXX analysis tool
my_pan=neu.openPan("XXX")
query items (hoover over the item to get the YYY name)
my_pan.get("YYY")
change item YYY
my_pan.set("YYY",value)
activate button ZZZ
my_pan.button("ZZZ")


In the window there are 3 tabs : Console, Script and Config .
You can execute a python script file by clicking on the icon in the menubar.

Console:

This is a python console.

Script:

Here you can write a script (to run it you have to press ctrl-return, command-return on Mac)

Config:

I the first line you can set the directory in which search for python scripts.
The second will set the python site path in which it will look for python modules.

The text box is the internal commands that are available to transform a nPhysD object to a 2D numpy array (and back): phys2array and array2phys.
