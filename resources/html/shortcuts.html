+ : increase zoom
- : decrease zoom
= : adapt zoom to window or set 100%
m : toggles mouse shape
r : show/hide origin ruler
g : show/hide grid
o : set origin to mouse position
Shift-o : same but for all images
h : open the horizontal lineout
v : open the vertical lineout
Option-Up : go to previous image
Option-Down : go to next image
Shift-C : open colorbar window
a : rescale image colors to limits
Option-Left : go to previous colorbar
Option-Right : go to next colorbar
> : increases the γ
< : reduces the γ Option-9 : Rescale the colorbar so that 99% of the pixels are visible
i : opens the list window
Shift-i : opens the measure window
Tab : cycle over paths
Once a path is selected, pressing 'w' will pop up the path-property window.
Arrows will move the path (Shift+arrow will move faster) and backspace will erase the path (if it's not owned by any window).
Lines:
b : toggle Bezier line c : toggle closed line p : prepend point a : append point d : delete point x : will put the first 2 point horizontal y : will put the first 2 point vertical
